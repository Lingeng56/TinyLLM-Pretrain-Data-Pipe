# 01 多源采集与融合（Guidance）

**要教会你的知识点**
- 多源数据接入与标准化的工程模式
- 数据来源的合规与许可记录
- 多源融合中的配额、权重与去重策略

**你需要完成的事情**
- 接入 3 类来源（公开语料 / 网页 / 对话数据）
- 为每个来源实现 connector 接口
- 输出统一 schema 的 raw 数据
- 生成来源清单与元数据

**建议步骤**
1) 选 3 类数据来源并明确许可
2) 为每个来源写 `connector_*.py`
3) 标准化输出 `data/raw/*.jsonl`
4) 生成 `reports/source_manifest.json`

**验收标准**
- 三类来源均可跑通
- raw 数据字段一致且可追溯
